# Dashboard

The **Dashboard** is your financial command center in Arthalekha. It provides a quick overview of your finances for any selected month, with visual summaries and key metrics.

## Overview

The dashboard shows:
- Monthly income and expense totals
- Net savings for the month
- Daily transaction breakdown
- Visual charts for quick understanding

## Accessing the Dashboard

The dashboard is the home page of Arthalekha. You'll see it immediately after logging in, or you can access it from the navigation menu.

## Dashboard Components

### Month Selector

At the top, you'll find controls to select which month to view:
- **Previous/Next buttons**: Navigate between months
- **Current month**: Quickly jump to the current month
- **Date display**: Shows the selected month and year

### Summary Cards

Quick metrics displayed prominently:

| Metric | Description |
|--------|-------------|
| **Total Income** | Sum of all incomes for the selected month |
| **Total Expenses** | Sum of all expenses for the selected month |
| **Net Savings** | Income minus Expenses (positive = saved, negative = overspent) |

### Daily Breakdown

A day-by-day view of the selected month showing:
- Date
- Income for that day
- Expenses for that day
- Running calculations

This helps you see:
- Which days had the most activity
- Spending patterns within the month
- Income timing

### Charts

Visual representations of your finances:

#### Income vs Expense Chart
- Bar chart comparing daily income and expenses
- Easy to spot high-spending days
- See income vs expense patterns

#### Balance Trend
- Line chart showing balance changes
- Track financial progress through the month

## Using the Dashboard

### Monthly Review

Use the dashboard for your monthly financial review:
1. Select the month you want to review
2. Check the summary cards for totals
3. Look at the charts for patterns
4. Drill into daily details if needed

### Quick Health Check

Glance at the dashboard to quickly understand:
- Are you in the green (saving) or red (overspending)?
- How does this month compare to typical months?
- Are there any unusual spikes?

### Pattern Recognition

Over time, using the dashboard helps you recognize:
- Typical monthly spending patterns
- High-expense periods
- Income timing and regularity

## Dashboard Data

### What's Included

The dashboard aggregates:
- All incomes for the selected month
- All expenses for the selected month
- Transfers are NOT included (they don't affect net worth)

### Real-time Updates

The dashboard reflects:
- All recorded transactions immediately
- Any edits or deletions
- Current state of your data

## Tips for Using the Dashboard

### Review Weekly

Check the dashboard at least weekly to:
- Stay aware of spending
- Catch any recording errors
- Maintain financial awareness

### Compare Months

Use the month selector to compare:
- This month vs last month
- Seasonal patterns
- Year-over-year trends

### Deep Dive from Dashboard

When you spot something interesting:
- Click through to detailed transaction lists
- Filter to understand specific categories
- Export for detailed analysis
